Introduction

Hello !

Bienvenue sur mon style guide pour mon application web Wild séries qui présentent les séries tv !

Dans cette page, je vais vous expliquer mes choix stylistiques pour créer cette application web

Le choix des couleurs

Pour les couleurs, j'ai décidé de m'orienter sur un code couleur sombre pour mettre en avant les affiches des séries tv présentés.

Et j'ai également choisi une couleur plus pétante pour le contraste et le pep's.

Le choix de la typo

J'ai choisi des typographies assez simples, sans empattement qui sont plus facile à lire, donc qui donnera une bonne expérience utilisateur.

Mais elles ont tout de même du caractère et ont une connotation qui ont un style assez marqué pour représenter l'impact visuel des séries tv.

La police choisie pour le logo et la connexion est imposante et attire le regard de l'utilisateur.

Voici les typo choisies :

Police utilisée pour Wild séries et Connexion : Sigmar One, taille 14

Police utilisée pour les titres de la nav : Changa, taille 13

Police utilisée pour les titres dans le contenu du site : Viga, taille 16
